# Offline TODO

> Offline TODO is a todo app for the browser using React JS that can be run on an offline basis and on hitting hard refresh also, the tasks or list remains intact on the page

## Technologies Used

- HTML5
- CSS3
- React JS
- React Hooks
- Font awesome
- Bootstrap
- Material UI
